# Action Bar

This is a simple desktop app that lets you quickly run actions and search for things.

## Try it out
Download the latest release from the [releases](https://github.com/joseph-tighe/action-bar/releases) page.

## Features
- Quickly run actions
- Search for things
- Customize the actions
- Customize the search
- Customize the UI
- Customize the shortcuts
- Customize the theme
